# AI Fitness Trainer Using MediaPipe: Squats Analysis

### Install required packages

```
pip install -r requirements.txt
```

### Execution

```
streamlit run demo.py
```

##### Run the app
```shell
docker run --rm -it \
-p 8080:8080 \
phongtn/squat-game:latest

```
